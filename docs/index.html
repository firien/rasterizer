Rasterizer



    Drag & Drop SVG image anywhere on this page.

    Conversion

    SVG file is used as an <img> src, and that is rendered to an internal <canvas>. From <canvas> the image is exported as a PNG. All files are packed into a zip file with jszip.

    About

    This is a PWA. It can be used offline, it does not communicate with any servers. All images stay on your machine, everything is done in your browser. Source code is available on GitHub.
